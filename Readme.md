*  功能
    将简单的EXCEL文件合并成一个CSV文件，使用c及vbscript开发，其中c部分使用mingw-gcc编译

* 使用方法：
[下载](https://github.com/JTaoZh/merge_csv_from_xls/releases/download/1.0/merge_csv_from_xls.zip)并解压

merge_csv_from_xls.bat [xlsx_folder] [output.csv]

* 使用建议：
将此文件夹加入至PATH路径（建议），且xlsx_folder下不能有tmp文件夹（必须）

* 举例：
merge_csv_from_xls.bat test test\output.csv
